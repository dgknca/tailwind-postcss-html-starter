Website Footprint Calculator

Pourquoi avons-nous conçu GreenWeb Index ? 

Quelle serait la meilleure méthode pour prendre conscience de l’impact environnemental d‘un site web ? Simplement en connaissant cet impact. Longtemps considéré comme inexistant, faute à l’analogie du « cloud », ce non-lieu quelque part dans le ciel, l’empreinte du numérique et plus précisément des services numériques est une réalité qu’il est nécessaire de rendre visible. Avec plus de 5000 sites testés, nous avons permis au plus grand nombre de se sensibiliser aux enjeux d’un numérique plus responsable. 

Qu’est-ce que
GreenWeb Index ? 

L’outil GreenWeb Index permet de tester l’impact environnementalmá d’une page de site web. Il a été conçu en combinant plusieurs briques technologiques Open Source qui permettent de collecter les données, et de les relier aux bonnes bases de données, à travers un algorithme de calcul.

L’indice GreenWeb

est calculé à partir de 5 critères principaux

Les données transférées au chargement d’une page

L’énergie utilisée par les Data centers, les réseaux télécoms et les terminaux

Le type de production énergétique utilisé

L’intensité carbone des énergies utilisées

L’audience du site web

Ces critères sont pondérés par

L’intensité énergétique des données web : 1.8kWh/GB

L’audience du site au mois en fonction du nombre moyen de pages vues 

L’intensité carbone des différentes sources énergétiques

  • 475 grammes CO2e par kWh
  • 33.4 grammes CO2e par kWh pour les énergies renouvelables
